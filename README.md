Описание



Простой калькулятор для командной строки, написанный на Python.



Поддерживает базовые арифметические операции и демонстрирует использование:



\- ввода данных из командной строки

\- условных операторов Python

\- простой структуры проекта



Требования

\- Python 3.10+

\- Windows / Linux / macOS

Использование



Запустите программу:



python app.py



Пример:



A: 10

Операция (+ - \* /): +

B: 5

Результат: 15
