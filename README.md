## Описание

**PyCal** — это минималистичный калькулятор, работающий в командной строке. Он выполняет четыре базовых арифметических действия и обрабатывает ошибку деления на ноль.

### Основные возможности

- Сложение (`+`)
- Вычитание (`-`)
- Умножение (`*`)
- Деление (`/`) с защитой от деления на ноль
- Понятные сообщения об ошибках (неверный оператор, деление на 0)
